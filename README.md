# Replication Project

## Paper

Balafoutas et al. (2024) – Competitiveness of Entrepreneurs and Salaried Workers

## Files

* data_final_cleaned.dta : dataset
* codes.do : replication code

## How to run

1. Download all files
2. Open Stata
3. Set working directory to this folder
4. Run:
   do codes.do

## Notes

Code reproduces the main tables and figures from the paper.
